# atlasOBEC
Atlas econômico da cultura brasileira (OBEC - UFRGS)

### Tecnologias utilizadas

 - PHP 7.0
 - D3JS (gráficos)
 - TopoJSON (coordenadas geográficas)
 - JQuery
 - HTML5
 - CSS3
 - Bootstrap (framework frontend)
 - CSV padronizado (Informações / Banco de Dados)
